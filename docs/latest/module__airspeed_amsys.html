Airspeed AMSYS (I2C)

Module XML file: airspeed_amsys.xml

Driver for a Amsys Differential Presure Sensor I2C (AMS 5812-0003-D)

Module configuration options

Define Options

  • name: AIRSPEED_AMSYS_I2C_DEV value: i2c0
    Description: change default i2c peripheral
  • name: AIRSPEED_AMSYS_OFFSET value: 0
    Description: sensor offset (default: 0)
  • name: AIRSPEED_AMSYS_SCALE value: 1.0
    Description: sensor scale factor (default: 1.0)
  • name: USE_AIRSPEED value: None
    Description: flag to use the data for airspeed control
  • name: SENSOR_SYNC_SEND value: None
    Description: flag to transmit the data as it is acquired

Periodic Functions

These functions are called periodically at the specified frequency from the module periodic loop.

    • Autorun: LOCK
