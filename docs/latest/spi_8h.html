enum SPISlaveSelect

SPI slave selection behavior.

SelectUnselect: slave is selected before transaction and unselected after Select: slave is selected before transaction but not unselected Unselect: slave is not selected but unselected after transaction NoSelect: slave is not selected nor unselected

Default operation should be SelectUnselected, but some peripherals might need some special control Use non-default control only if you know what you're doing
