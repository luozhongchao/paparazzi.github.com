Detailed Description

Driver for the analog to digital converters on STM32.

Usage: Define flags for ADCs to use and their channels:

-DUSE_AD1 -DUSE_AD1_1 -DUSE_AD1_3

would enable ADC1 and it's channels 1 and 3.
